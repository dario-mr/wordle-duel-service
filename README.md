# Wordle Duel service

Backend service for the Wordle Duel game.

## Env variables

- `PORT`: server port (default: `8088`)
- `PROFILE`: spring profile (default: `prod`)
- `DB_PORT`: db port (default: `null`)
- `DB_USER`: db user (default: `null`)
- `DB_PASSWORD`: db password (default: `null`)

For a sample env file, see [.env.example](.env.example).
